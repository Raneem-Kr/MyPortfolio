Home Projekte Über mich Kotakt

Hier sind einige Websites, die ich selbst entwickelt habe:

Diabetes ist eine Website, die umfassende Informationen über die Krankheit Diabetes bietet. Für die Entwicklung der Seite wurden HTML, CSS (einschließlich Bootstrap) und JavaScript (React) verwendet. Zudem ist die Website vollständig responsiv gestaltet und passt sich verschiedenen Bildschirmgrößen an.

Moso Interior ist eine Website für Möbel und Sofas, die mit HTML, CSS und Tailwind CSS entwickelt wurde. Die Seite ermöglicht es Nutzern, Produkte hinzuzufügen und gezielt nach bestimmten Artikeln zu suchen. Sie ist responsiv gestaltet und bietet ein ansprechendes, benutzerfreundliches Design.

Simple Café ist eine Website für ein Café, die mit HTML, CSS und Tailwind CSS entwickelt wurde. Die Seite präsentiert verschiedene Kaffeespezialitäten und Getränke. Die Website ist vollständig responsiv gestaltet und passt sich an verschiedene Bildschirmgrößen an.

The Path Church ist eine Website, die als Übungsprojekt mit HTML, CSS und Tailwind CSS entwickelt wurde. Die Seite ist vollständig responsiv gestaltet und passt sich verschiedenen Bildschirmgrößen an.
